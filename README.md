# Psi_Devboard2040

A custom built Devboard based of Rasberry-Pi RPI2040 microcontroller. Built with a focus on effeciency and simplicity. Provides as solid tool for future embedded systms project in C/C++ or Python. Very valuble learning experience. The most complex PCB project to date. This project was done as part of a larger electronics and robotics competition #Stasis. 


**This project is named after the greek letter Psi**
